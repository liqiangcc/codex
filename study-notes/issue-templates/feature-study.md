# Feature Study Issue Template

用于创建个人学习 issue。标题格式：

```text
[Feature Study] 功能名称
```

建议标签：

```text
study, feature-map, stage-1
```

## 摘要

- 状态：Todo
- 一句话说明：
- 当前目标：
- 当前结论：
- 下一步：
- 详细文档：

## 当前任务

- [ ] 阅读详细文档。
- [ ] 完成最小练习。
- [ ] 记录一个观察或问题。
- [ ] 更新详细文档、实验记录或功能索引。

## 本轮记录

- 暂无记录。

## 待验证问题

- 写具体问题，不写泛泛占位。

## 完成条件

- [ ] 能用一句话解释该功能。
- [ ] 写清适用场景、边界、约束和风险。
- [ ] 完成至少一个最小练习或真实任务。
- [ ] 记录验证方式和结果。
- [ ] 详细文档中的当前结论已更新。
- [ ] `feature-map.md` 状态已同步。
